CSS Animations

There are 2 components of an animation:

  1. style describing animation
  2. a set of keyframes describing start/end/midpoints

Advantages of using CSS (rather than Javascript)

  • Easy to use for easy animations
  • Runs well in browser (allowing it skip frames as necessary, etc.)
  • Allows browser to optimize, rather than requiring it consistently (such as when you are on a different tab)

There are a number of basic properties that can help you get your animations going:

  • name: this is the name of your @keyframes
  • duration: default = 0s (no animation)
  • iteration-count
  • delay (negative values allowed)
  • direction (options are normal (default), reverse, alternate, and alternate-reverse)
  • timing-function
  • fill-mode: what happens before and after the animation?
  • play-state
Timing functions:

Linear: same speed

Ease: default

Ease-in: slow start

Ease-out: slow end

Ease-in-out: slow start / end
